Ex: Input = ["Ryan", "Kieran", "Jason", "Yous"], Output = ["Ryan", "Yous"]

Note: keep the original order of the names in the output.

friend(["Ryan", "Kieran", "Mark"]), ["Ryan", "Mark"]
